## 1. 핵심 디자인 전략

### 1.1 3가지 핵심 원칙

| 원칙 | 설명 | 구현 방법 |
|------|------|----------|
| **친근함 + 신뢰성** | 고양이 캐릭터로 접근성 높이되, 정확한 정보로 신뢰 구축 | 캐릭터 일러스트 + 만세력 강조 |
| **명확함** | 어려운 사주 용어 배제, 일상 언어로 쉬운 이해 | 심플한 카드 레이아웃 + 쉬운 표현 |
| **공유 가능성** | 인스타그램에 공유하고 싶은 예쁜 결과 카드 | 골드/로즈골드 악센트 + 고급감 |

### 1.2 경쟁사 분석 결과

**포스텔러 vs 팔자냥:**
- 포스텔러: 컬러풀, 놀이감 → 팔자냥: 럭셔리한 톤, 프리미엄 느낌
- 포스텔러: 여성 캐릭터 → 팔자냥: 고양이 (중성적, 광범위한 타겟)

**점신 vs 팔자냥:**
- 점신: 과도한 광고, 올드한 디자인 → 팔자냥: 광고 없음, 모던 미니멀
- 점신: 무료 모델 → 팔자냥: ₩1,500 저가 (프리미엄 느낌)

**헬로우봇 vs 팔자냥:**
- 헬로우봇: 채팅형, 정확도 낮음 → 팔자냥: 카드형, 만세력 기반 신뢰성

## 2. 색상 팔레트

### 2.1 선택된 색상 (Option 1: 우아한 자주색 + 골드)

```
Primary Color (주색)
├─ Deep Lavender: #6B5B95 (주요 버튼, 헤더, 강조 요소)
└─ Soft Purple: #9B8FB7 (서브 강조, 호버 상태)

Accent Color (포인트색)
├─ Gold: #D4AF37 (특별 콘텐츠, 프리미엄 표시)
└─ Rose Gold: #B76E79 (궁합, 부드러운 강조)

Background
├─ Off-White: #F9F7F4 (메인 배경, 따뜻함)
├─ Cream: #FAF8F3 (카드 배경)
└─ White: #FFFFFF (모달, 입력폼)

Text
├─ Deep Charcoal: #2C2C2C (본문)
├─ Medium Gray: #6B6B6B (보조 텍스트)
└─ Light Gray: #A0A0A0 (비활성/플레이스홀더)

Five Elements (오행)
├─ 목(木): #7FB069 (Green)
├─ 화(火): #FF6B6B (Red)
├─ 토(土): #FFB366 (Orange)
├─ 금(金): #C0C0C0 (Silver)
└─ 수(水): #4ECDC4 (Teal)
```

### 2.2 색상 심리학

- **자주색**: 영적/신비로움 (사주 서비스와 상성 좋음)
- **골드**: 럭셔리 + 여성스러움 (₩1,500이지만 고급감)
- **따뜻한 톤**: MZ 여성의 미니멀 + 따뜻함 선호

## 3. 타이포그래피

### 3.1 폰트 전략

| 계층 | 폰트 | 크기 | 굵기 | 용도 |
|------|------|------|------|------|
| Display | Noto Serif KR | 32-40px | Bold | 로고, 메인 캐피 |
| H1 | Pretendard | 24-28px | Bold | 페이지 제목 |
| H2 | Pretendard | 20-24px | SemiBold | 섹션 제목 |
| H3 | Pretendard | 18px | SemiBold | 카드 제목 |
| Body Large | Pretendard | 16px | Regular | 중요 본문 |
| Body | Pretendard | 14px | Regular | 일반 본문 |
| Small | Pretendard | 13px | Regular | 부가 정보 |
| Caption | Pretendard | 12px | Regular | 극소 텍스트 |

### 3.2 폰트 선택 이유

- **Pretendard**: 웹 최적화, 무료 오픈소스, 가독성 우수
- **Noto Serif KR**: 우아함 (로고/디스플레이용)

## 4. 컴포넌트 라이브러리

### 4.1 Button (3가지 Variant)

```
Primary Button (주요 CTA)
├─ Background: Deep Lavender (#6B5B95)
├─ Hover: #5A4A84
├─ Padding: 16px (수직), 24px (수평)
├─ Height: 48px (최소 터치 타겟)
└─ Border Radius: 8px

Secondary Button (보조)
├─ Style: Outlined (2px border)
├─ Border Color: Soft Purple (#9B8FB7)
├─ Hover: Light Lavender 배경
└─ 용도: 취소, 대안 액션

Ghost Button (링크형)
├─ Background: Transparent
├─ Text Color: Medium Gray → Deep Lavender (호버)
└─ 용도: 더보기, 취소 (최소 강조)
```

### 4.2 Input Field

```
크기: 48px (최소 터치 타겟)
배경: White
테두리: 1px #E0E0E0
포커스:
  - 테두리 색 변경 → #6B5B95
  - 박스 섀도우 추가: 0 0 0 3px rgba(107, 91, 149, 0.1)
  - 전환: 150ms
상태:
  - Normal: 흰 배경
  - Focus: 자주색 테두리 + 섀도우
  - Error: 빨간 테두리 + 빨간 배경 (light)
  - Disabled: 회색 배경 + 비활성 커서
```

## 5. 페이지별 UX 흐름

### 5.1 홈 화면 (`/home`)

**목표:** 4개 사주 유형 중 선택, 심플하고 명확함

```
┌─ Header (Fixed) ──────────────────┐
│ 팔자냥    [코인: 3.5]   [프로필]  │
├────────────────────────────────────┤
│ 안녕하세요, OOO님! 🐱              │
├────────────────────────────────────┤
│ [탭] 개인사주 신년운세 궁합 연애운  │
├────────────────────────────────────┤
│ ┌────────────────────────────────┐ │
│ │ 🔮 개인 사주                   │ │ (Card)
│ │ 설명...                        │ │
│ │ • 성격과 기질                  │ │
│ │ • 타고난 재능                  │ │
│ │ • 대운 흐름                    │ │
│ │ [사주 보러가기 (1코인)]         │ │
│ └────────────────────────────────┘ │
├────────────────────────────────────┤
│ 🎁 공유하면 1코인 무료!             │
└────────────────────────────────────┘
```

**UX 포인트:**
- 상단 헤더: 고정 (sticky)
- 4개 탭: 하나만 활성 (라운드 밑줄)
- 사주 유형 카드: 호버 시 그림자 증가
- 상호작용성: 명확한 CTA 버튼

### 5.2 정보 입력 (`/input/:type`)

**목표:** 필수 정보만 명확하게 수집

```
┌─ Header ──────────────────────────┐
│ ◀ 개인 사주               설정 ●●● │
├────────────────────────────────────┤
│ 누구의 사주를 볼까요?               │
│ [본인] [연인] [가족] [기타] (토글) │
├────────────────────────────────────┤
│ 생년월일 * (필수)                  │
│ [년도] [월] [일]    (스크롤 선택)  │
│ ○ 양력  ● 음력  ○ 윤달  (라디오)  │
├────────────────────────────────────┤
│ 태어난 시간 (선택)                 │
│ [드롭다운: 오후 2시]               │
│ ☑ 모르겠어요 (체크)                │
├────────────────────────────────────┤
│ 성별 *                             │
│ [남성] [여성] (토글)               │
├────────────────────────────────────┤
│ [내 만세력 확인하기] (Primary Btn) │
└────────────────────────────────────┘
```

**UX 포인트:**
- 필드 라벨: 명확한 색상 구분
- 입력값 검증: 포커스 시 테두리 색 변경
- 선택 우선: 라디오/토글로 한 번에 선택
- 다음 버튼: 모든 필수 값 입력 후 활성화

### 5.3 결제 유도 (`/preview`)

**목표:** 무료 미리보기로 관심 유발, 자연스러운 전환

```
┌─ Header ──────────────────┐
│ ◀ 내 만세력                │
├─────────────────────────────┤
│ ┌─ Highlighted Card ────┐  │
│ │ 🐱 ✨                 │  │
│ │ OOO님의 사주팔자       │  │
│ │ [년주] [월주]...      │  │
│ │ 일간: 戊土 (무토)      │  │
│ └───────────────────────┘  │
├─────────────────────────────┤
│ 오행 분포                    │
│ 木 ██░░░ 20%               │
│ 火 ████░ 40%               │
│ ...                        │
├─────────────────────────────┤
│ ┌─ 블러 처리 ────────────┐  │
│ │ 🔒 상세 해석 미리보기   │  │
│ │ ░░░░░░░░░░░░░░░░░░  │  │
│ │ (블러)  전체 해석 보기→│  │
│ └────────────────────────┘  │
├─────────────────────────────┤
│ 보유 코인: 0 🪙              │
│ [🔮 전체 해석 보기 (1코인)]  │
│ (또는)                       │
│ [💰 코인 충전하고 보기]      │
└─────────────────────────────┘
```

**UX 포인트:**
- 사주팔자 미리보기: 강조된 카드 (Highlighted variant)
- 블러 처리: 심리적 기대감 생성
- 이중 CTA: 코인 있으면 "전체 해석", 없으면 "충전"
- 점진적 공개: 불안감 없이 자연스러운 결제 유도

### 5.4 결과 화면 (`/result/:id`)

**목표:** 결과를 예쁘게 표현, 공유 욕구 극대화

```
├─────────────────────────────┤
│ ┌─ Highlighted Card ────┐  │
│ │ OOO님의 사주           │  │

├─────────────────────────────┤

│ 당신은 화의 기운이 강한...   │
│ [더보기 ▼]                 │
├─────────────────────────────┤
│ 🔮 상세 분석 (아코디언)    │
│ ▼ 성격과 기질 (펼침)        │
│   [콘텐츠]                 │
│ ▼ 적성과 진로 (접기)        │

│ [📥 이미지 저장]            │
└─────────────────────────────┘
```

**UX 포인트:**
- 결과 카드: 강조된 스타일로 공유 욕구 자극
- 아코디언: 첫 항목만 펼쳐짐 (progressive disclosure)
- 공유 버튼: 다양한 방식 제공 (확산 최대화)
- 시각화: 오행 차트로 한눈에 이해 가능

## 6. 차별화 전략

### 6.1 포스텔러 대비 차별화

| 요소 | 팔자냥 전략 | 이유 |
|------|-----------|------|
| 색상 톤 | 럭셔리한 자주색 + 골드 | 포스텔러보다 성숙한 이미지 |
| 레이아웃 | 심플 + 카드 중심 | 명확성과 집중도 높음 |
| 캐릭터 | 고양이 (중성적) | 더 광범위한 타겟 수용 |
| 디자인 언어 | 프리미엠한 미니멀 | 가성비 서비스지만 고급감 |
| 타이포그래피 | Pretendard (현대적) | 더 신뢰스러운 느낌 |

### 6.2 점신 대비 차별화

| 요소 | 팔자냥 전략 | 이유 |
|------|-----------|------|
| 광고 | 없음 | 이탈 최소화 |
| 디자인 | 모던 미니멀 | 올드함 배제 |
| 타겟 | MZ 여성 명확 | 포지셔닝 명확 |
| 성능 | 빠른 로딩 | 가벼운 인터페이스 |

### 6.3 헬로우봇 대비 차별화

| 요소 | 팔자냥 전략 | 이유 |
|------|-----------|------|
| 인터페이스 | 카드형 (명확) | 채팅보다 정보 구조화 |
| 신뢰성 | 만세력 기반 강조 | 정확도 차별화 |
| 디자인 | 럭셔리한 톤 | 더 신뢰스러운 느낌 |
| 가격 | ₩1,500 | 합리적 가격대 명시 |

### 6.4 고양이 캐릭터 활용

```
팔자냥 (Paljjanyang) 캐릭터

용도:
1. 로고: 텍스트 "팔자냥" 옆에 32px
2. 로딩: 중앙 64px, 애니메이션 (bounce)
3. 결과 카드: 좌상단 48px
4. 빈 상태: 96px (no data)
5. 성공 메시지: 40px 아이콘

표정:
├─ 기본: 평정 (신뢰성)
├─ 로딩: 생각 중 + 애니메이션
├─ 성공: 웃음 + 반짝임
└─ 안내: 친근한 포즈

색상: Deep Lavender (#6B5B95)
스타일: 미니멀 + 귀여움
```

### 팔자냥의 비주얼 아이덴티티

```
색상: 자주색(#6B5B95) + 골드(#D4AF37)
     → 럭셔리 + 현대성 + 여성스러움

타입: Pretendard + Noto Serif KR
     → 신뢰성 + 우아함

레이아웃: 심플 + 카드 중심
         → 명확함 + 집중도

캐릭터: 고양이 🐱
        → 친근함 + 신뢰성

가격감: ₩1,500이지만 프리미엄
       → 가성비 최고의 서비스
```

### 경쟁 우위 (Competitive Advantage)

1. **디자인**: 포스텔러보다 성숙하고 럭셔리함
2. **신뢰성**: 점신보다 명확하고 구체적인 정보
3. **가격**: ₩1,500 저가 (헬로우봇과 유사하지만 더 신뢰성 높음)
4. **타겟**: MZ여성 명확 (나이, 성별, 라이프스타일)
5. **경험**: 광고 없음, 빠름, 쉬움
