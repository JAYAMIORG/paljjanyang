# 팔자냥 화면 설계 (Wireframe)

> 작성일: 2025-01-15
> 플랫폼: 모바일 웹 (반응형)

---

## 1. 화면 목록

| # | 화면명 | 설명 | 인증 필요 |
|---|--------|------|-----------|
| 1 | 랜딩 페이지 | 서비스 소개, CTA | X |
| 2 | 로그인 | 소셜 로그인 + 이메일 로그인 (메인 인증) | X |
| 3 | 회원가입 | 이메일 회원가입 (소셜은 자동 가입) | X |
| 4 | 홈 | 사주 유형 선택 (4개 탭) | O |
| 5 | 정보 입력 | 생년월일시 등 입력 | O |
| 6 | 결제 유도 | 만세력 미리보기 + 결제 유도 | O |
| 7 | 결제 | 코인 패키지 선택 | O |
| 8 | 로딩 | 사주 분석 중 | O |
| 9 | 결과 | 사주 결과 표시 | O |
| 10 | 마이페이지 | 과거 기록, 설정 | O |

## 4. 컴포넌트 정의

### 4.1 공통 컴포넌트

| 컴포넌트 | 용도 |
|----------|------|
| `Header` | 상단 네비게이션 (뒤로가기, 타이틀, 프로필) |
| `Button` | Primary, Secondary, Ghost 버튼 |
| `Input` | 텍스트 입력 필드 |
| `Select` | 드롭다운 선택 |
| `DatePicker` | 생년월일 선택 |
| `TimePicker` | 시간 선택 (시주) |
| `Card` | 사주 유형 카드, 결과 카드 |
| `Modal` | 팝업 모달 |
| `Toast` | 알림 메시지 |
| `Loading` | 로딩 스피너/애니메이션 |
| `Chart` | 오행 차트 시각화 |

### 4.2 페이지별 고유 컴포넌트

| 컴포넌트 | 페이지 | 용도 |
|----------|--------|------|
| `SajuTypeCard` | 홈 | 사주 유형 선택 카드 |
| `SajuTypeTabs` | 홈 | 4개 탭 네비게이션 |
| `CoinBalance` | 홈, 마이페이지 | 코인 잔액 표시 |
| `PersonInfoForm` | 정보 입력 | 생년월일시/성별 입력 폼 |
| `SajuPreview` | 결제 유도 | 사주팔자 미리보기 카드 |
| `FiveElementChart` | 결제 유도, 결과 | 오행 분포 차트 |
| `CoinPackageCard` | 결제 | 코인 패키지 선택 카드 |
| `ResultSummary` | 결과 | 핵심 요약 섹션 |
| `ResultDetail` | 결과 | 아코디언 형태 상세 분석 |
| `MonthlyFortune` | 신년운세 결과 | 월별 운세 카드 |
| `ShareCard` | 결과 | 공유용 이미지 카드 |
| `HistoryItem` | 마이페이지 | 과거 사주 기록 아이템 |

## 5. 라우팅 구조

```
/                   # 랜딩 페이지
/signup             # 회원가입
/login              # 로그인
/home               # 홈 (사주 유형 선택 - 4개 탭)
/input/:type        # 정보 입력 (type: personal, yearly, compatibility, love)
/preview            # 결제 유도 (만세력 미리보기)
/payment            # 결제 (코인 충전)
/loading            # 로딩
/result/:id         # 결과
/mypage             # 마이페이지
/mypage/history     # 사주 기록
/mypage/settings    # 설정
```
