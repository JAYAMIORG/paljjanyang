# 팔자냥 디자인 시스템 & 가이드라인

> 버전: 1.0
> 작성일: 2025-01-16
> 목표: MZ세대 여성 타겟, 포스텔러 참고 + 현대적 미니멀 디자인

---

## 1. 디자인 철학

### 1.1 핵심 원칙

1. **친근함 + 신뢰성** - 고양이 캐릭터로 접근성 높이되, 정확한 정보로 신뢰 구축
2. **명확함** - 어려운 사주 용어 배제, 일상 언어로 쉬운 이해
3. **공유 가능성** - 인스타그램에 공유하고 싶은 예쁜 결과 카드
4. **모바일 중심** - 모바일 웹 최적화, 터치 친화적 UX
5. **현대성** - 2025-2026 트렌드 반영 (Glassmorphism, 마이크로인터랙션)

### 1.2 타겟 사용자 특성

| 특성 | 특징 | 디자인 반영 |
|------|------|-----------|
| MZ세대 | 디지털 네이티브, SNS 활용 활발 | 공유 최적화, 모던 인터페이스 |
| 여성 | 감각적, 큐레이션 선호 | 컬러풀하면서도 세련된 톤 |
| 20-30대 | 직장인/학생, 자기계발 관심 | 실용적, 액셔너블한 조언 |
| 가성비 중심 | ₩1,500 저가 서비스 기대 | 프리미엄 느낌이면서도 가볍고 빠른 경험 |

## 2. 색상 팔레트 (Color System)

### 2.1 추천 색상 조합 (3가지 옵션)

#### Option 1: "우아한 자주색 + 골드" (권장)
**테마:** 전통 + 현대성, 여성스러우면서도 럭셔리한 느낌

```
Primary (주색)
- Deep Lavender: #6B5B95 (주요 버튼, 헤더, 강조 요소)
- Soft Purple: #9B8FB7 (서브 강조, 호버 상태)

Accent (포인트색)
- Gold: #D4AF37 (특별 콘텐츠, 프리미엄 표시, 별점)
- Rose Gold: #B76E79 (부드러운 강조, 궁합 테마)

Background
- Off-White: #F9F7F4 (메인 배경, 따뜻하고 부드러운 톤)
- Cream: #FAF8F3 (카드 배경)
- White: #FFFFFF (모달, 입력폼 배경)

Text
- Deep Charcoal: #2C2C2C (본문 텍스트)
- Medium Gray: #6B6B6B (서브 텍스트)
- Light Gray: #A0A0A0 (비활성, 플레이스홀더)
```

**사용 예시:**
- Primary Button: Deep Lavender (#6B5B95)
- Hover State: Soft Purple (#9B8FB7)
- Special Badge (무료 리워드): Gold (#D4AF37)
- Compatibility Result: Rose Gold (#B76E79)
- Page Background: Off-White (#F9F7F4)

**적용 이유:**
- 자주색: 영적/신비로운 느낌 (사주 서비스와 상성 좋음)
- 골드: 럭셔리하면서도 여성스러움 (포스텔러도 유사 전략)
- 따뜻한 톤: MZ 여성의 미니멀 + 따뜻함 선호

### 2.2 권장 선택: Option 1 (Deep Lavender + Gold)

**선택 이유:**
1. 사주/운세 서비스와 색상 심리 일치 (신비로움 + 고급스러움)
2. 포스텔러와 차별화 (포스텔러는 더 밝고 컬러풀)
3. 럭셔리하면서도 친근함 (팔자냥 브랜딩 전략과 맞음)
4. 다크모드 확장성 우수
5. WCAG AAA 대비도 충족 가능

## 3. 타이포그래피 (Typography)

### 3.1 한글 폰트 추천

#### 주요 폰트 (3가지 조합)

**조합 A: 우아함 + 가독성 (권장)**

```
Display (로고, 초대형 제목)
- 폰트명: 이루 (Iru) / 배달의민족 을지로체
- 크기: 28px ~ 40px
- 특징: 따뜻함, 차별화된 느낌
- 용도: 로고, 캐릭터 이름 "팔자냥"

Heading (페이지 제목)
- 폰트명: 간달테공체 / Pretendard Bold
- 크기: 20px ~ 24px
- 특징: 현대적, 강한 가독성
- 용도: 화면 제목, 카드 제목

Subheading (섹션 제목, 강조)
- 폰트명: Pretendard SemiBold
- 크기: 16px ~ 18px
- 용도: 섹션 제목, 중요 정보

Body (본문, 설명 텍스트)
- 폰트명: Pretendard Regular / 맑은 고딕
- 크기: 14px ~ 16px
- Line Height: 1.6 (넉넉한 줄간격)
- Letter Spacing: 0.3px (약간의 자간 확대)
- 용도: 사주 해석, 일반 본문

Caption (보조 텍스트, 메타정보)
- 폰트명: Pretendard Regular
- 크기: 12px ~ 13px
- Color: Medium Gray (#6B6B6B)
- 용도: 날짜, 부가 정보, 플레이스홀더
```

**선택 이유:**
- Pretendard: 웹 최적화, 무료 오픈소스, 가독성 우수
- 이루/을지로체: 팔자냥만의 아이덴티티 (로고, 브랜드 텍스트용)

**구현 방법 (Next.js + TailwindCSS):**

```html

/* Display font (Google Fonts 유사체) */
@import url('https://fonts.googleapis.com/css2?family=Noto+Serif+KR:wght@400;700&display=swap');
```


1. **상단 헤더 (Fixed, 고정)**
   - 높이: 56px
   - 배경: White with subtle shadow
   - 로고/브랜드명 왼쪽 정렬
   - 코인, 프로필 아이콘 오른쪽 정렬

3. **사주 유형 카드**
   - 카드 배경: Cream (#FAF8F3)
   - 테두리: 1px, Light Gray (#E8E8E8)
   - Border Radius: 12px
   - 패딩: 24px (모바일), 32px (태블릿+)

   - 제목: 18px, Bold
   - 설명: 14px, Regular, 줄간격 1.6
   - 버튼: Primary color (Deep Lavender), 16px
   - 호버 효과: 약간의 그림자 + 배경 색 변화
   - 터치 피드백: Active state 시 opacity 0.9

4. **리워드 배너**
   - 배경: Light Gold (#FFF8E6)
   - 아이콘: 🎁 (또는 선물 일러스트)
   - 텍스트: 16px, SemiBold, Deep Lavender
   - 위치: 페이지 하단 sticky
   - 높이: 60px + 패딩

## 12. 경쟁사 대비 차별화 요소

### 12.1 포스텔러 vs 팔자냥

| 항목 | 포스텔러 | 팔자냥 | 차별점 |
|------|---------|--------|--------|
| 색상 | 컬러풀 (5+ 색상) | 일관성 있는 톤 (3색) | 럭셔리 + 미니멀 |
| 캐릭터 | 귀여운 여캐 | 고양이 (중성적) | 더 광범위한 타겟 |
| 디자인 | 놀이감 느낌 | 프리미엄 느낌 | 가성비 + 고급감 |
| 레이아웃 | 복잡한 콘텐츠 | 심플 + 명확 | 빠른 이해 |
| 가격 | 높음 | 최저가 (₩1,500) | 접근성 |

### 12.2 점신 vs 팔자냥

| 항목 | 점신 | 팔자냥 |
|------|------|--------|
| 광고 | 과도 (이탈의 주요 원인) | 광고 없음 (유료 모델) |
| 디자인 | 올드해 보임 | 모던 미니멀 |
| 타겟 | 전연령 | MZ 여성 명확 |
| 가격 | 무료 + 광고 | ₩1,500 저가 |

### 12.3 헬로우봇 vs 팔자냥

| 항목 | 헬로우봇 | 팔자냥 |
|------|---------|--------|
| 인터페이스 | 채팅형 | 카드형 (명확) |
| 정확도 | 논란 있음 | 만세력 기반 강조 |
| 디자인 | 캐주얼 | 프리미엄 |
| 신뢰성 | 낮음 | 정확도 강조 |
