# Canva 작업 가이드 - 팔자냥 인스타그램

## Quick Start (5분 요약)

1. [canva.com](https://www.canva.com) 접속 → 로그인
2. "인스타그램 게시물" 검색 → 템플릿 선택
3. 브랜드 색상 적용: `#6B5B95` (보라), `#F9F7F4` (배경)
4. 캐릭터 이미지 업로드: `assets/brand-character.png`
5. `COPY_TEXT.txt` 파일에서 텍스트 복사하여 붙여넣기
6. PNG로 다운로드 → 인스타그램 업로드

---

## Step 1: Canva 브랜드 키트 설정

### 1.1 브랜드 키트 접근
1. Canva 좌측 메뉴 → **브랜드**
2. **브랜드 키트** 클릭
3. (무료 계정은 1개만 가능)

### 1.2 색상 추가
**+ 추가** 버튼을 눌러 다음 색상 입력:

| 이름 | HEX 코드 |
|------|----------|
| Primary | `#6B5B95` |
| Gold | `#D4AF37` |
| Rose Gold | `#B76E79` |
| Background | `#F9F7F4` |
| Text | `#2C2C2C` |

### 1.3 폰트 설정
- **한글**: Noto Sans KR (Canva에서 제공)
- **영문**: Poppins 또는 Inter

---

## Step 2: 템플릿 선택

### 2.1 새 디자인 만들기
1. 홈 → **디자인 만들기**
2. 검색창에 "인스타그램 게시물" 입력
3. **Instagram 게시물 (정사각형)** 선택 - 1080 x 1080px

### 2.2 템플릿 찾기
좌측 **템플릿** 탭에서:
- 검색어: "minimal", "modern", "simple"
- 추천 스타일: 미니멀, 플랫 디자인, 그라디언트

### 2.3 추천 템플릿 유형
| 슬라이드 | 추천 템플릿 스타일 |
|----------|-------------------|
| 1장 (표지) | 그라디언트 배경 + 큰 텍스트 |
| 2-4장 | 흰 배경 + 카드형 레이아웃 |
| 5장 (CTA) | 단색 배경 + 중앙 텍스트 |

---

## Step 3: 이미지 업로드

### 3.1 업로드 방법
1. 좌측 메뉴 → **업로드**
2. **파일 업로드** 클릭
3. `marketing/instagram/assets/` 폴더에서 선택:
   - `brand-character.png` (필수)
   - `logo-opened.png` (선택)

### 3.2 이미지 배치 팁
- 캐릭터 이미지는 **배경 제거** 기능 활용 (Pro 기능)
- 무료 계정: 이미 PNG 배경이 투명함
- 크기: 슬라이드의 30-50% 정도

---

## Step 4: 각 슬라이드 제작

### 슬라이드 1: 표지

**배경 설정:**
1. 배경 클릭 → 좌측 **배경** 탭
2. **그라디언트** 선택
3. 색상 1: `#6B5B95`
4. 색상 2: `#9B8FB7` (연한 보라)

**텍스트 추가:**
```
메인 (큰 글씨): ₩1,500으로 보는 AI 사주
- 폰트: Noto Sans KR Bold
- 크기: 60-80px
- 색상: #FFFFFF

서브 (작은 글씨): 팔자냥 🐱
- 폰트: Noto Sans KR Medium
- 크기: 30-40px
- 색상: #FFFFFF
```

**캐릭터 배치:**
- 하단 또는 우측에 캐릭터 이미지 배치
- 크기: 높이 40% 정도

---

### 슬라이드 2: 서비스 소개

**배경:** `#F9F7F4` (Off-White)

**텍스트:**
```
제목: 팔자냥이 뭐야?
- 폰트: Noto Sans KR Bold
- 크기: 50px
- 색상: #6B5B95

본문:
• AI가 분석하는 사주팔자
• 개인사주 / 신년운세 / 궁합 / 연애운
• 전문가 수준의 해석을 1,500원에
- 폰트: Noto Sans KR Regular
- 크기: 24-28px
- 색상: #2C2C2C
- 줄간격: 1.5
```

---

### 슬라이드 3: 핵심 기능

**배경:** `#F9F7F4`

**레이아웃:** 2x2 그리드

**텍스트 (각 그리드):**
```
🔮 개인사주          📅 신년운세
타고난 성격과 운명    올해 월별 운세

💕 궁합              ❤️ 연애운
두 사람의 케미 분석   이상형과 연애 스타일
```

**팁:**
- 각 기능을 카드 형태로 구분
- 이모지는 크게 (40px+)
- 설명은 작게 (20px)

## Step 5: 다운로드

### 5.1 다운로드 설정
1. 우측 상단 **공유** → **다운로드**
2. 파일 형식: **PNG**
3. 품질: **높음** 또는 **최고 품질**

### 5.2 다중 페이지 다운로드
- "모든 페이지" 선택
- 또는 개별 페이지 선택하여 다운로드
- 파일명: `palzza_carousel_1.png` ~ `palzza_carousel_5.png`

---

## Step 6: 인스타그램 게시

### 6.1 인스타그램 앱에서 직접 게시
1. 인스타그램 앱 열기
2. **+** 버튼 → **게시물**
3. 5장 이미지 **순서대로** 선택
4. 필터: **없음** (원본 유지)
5. **다음** → 캡션 입력
6. `COPY_TEXT.txt`에서 캡션 복사
7. **공유**

### 6.2 Meta Business Suite (예약 게시)
1. [business.facebook.com](https://business.facebook.com) 접속
2. 인스타그램 계정 연결
3. **콘텐츠** → **게시물 만들기**
4. 이미지 업로드 + 캡션 작성
5. 예약 시간 설정 (선택)
6. **게시** 또는 **예약**

## 트러블슈팅

### Q: 한글 폰트가 깨져요
- Canva 기본 제공 폰트 사용: Noto Sans KR

### Q: 이미지가 흐려요
- 다운로드 시 "최고 품질" 선택
- 원본 이미지가 고해상도인지 확인

### Q: 카로셀 순서가 바뀌었어요
- 인스타그램 앱에서 이미지 선택 시 순서 확인
- 첫 번째 선택한 이미지가 첫 장이 됨
